# Evidence (portfolio)

This folder is for screenshots, logs, and “proof artifacts” that demonstrate outcomes
(e.g., security findings, pipeline runs, policy enforcement, dashboards).

## What belongs here
- Screenshots of key AWS console views (redacted)
- Example findings (JSON snippets, sanitized exports)
- CI/CD evidence (pipeline logs, scan results)
- Cost estimation snapshots (assumptions + outputs)

## Redaction rules
- Remove account IDs, emails, access keys, secrets
- Blur internal hostnames/IPs if needed
- Replace sensitive values with placeholders

## Suggested naming
- `evidence-<area>-<what>-YYYYMMDD.png`
  مثال: `evidence-securityhub-findings-20260218.png`
